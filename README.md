# MAIM Legacy Media Studio

**MAIM Legacy Media Studio** is the production layer that turns human wisdom, personal archives, long-form thought, and source-backed knowledge into living media assets.

It sits beside the **Legacy Media Vault**:

- **Legacy Media Vault** preserves, organizes, enriches, and makes old media searchable.
- **MAIM Legacy Media Studio** transforms that intelligence into podcasts, interviews, videos, clips, books, courses, publishing packs, avatar content, and future legacy products.

## Core Thesis

AI will make content abundant. Real lived wisdom will become the differentiator.

MAIM Legacy Media Studio exists to help wisdom holders — athletes, founders, elders, healers, teachers, artists, operators, and creators — turn what they know into structured, distributable, monetizable IP.

> Give the system long-form wisdom once. It should be able to help turn it into a living media estate.

## Primary System Relationship

```text
Legacy Media Vault = memory + archive intelligence
NotebookLM = synthesis brain
MAIM Legacy Media Studio = production + expression engine
maim-legacy-media-skills = reusable workflow library
AMA Mob / HAMAL = broader agent ecosystem that must be able to discover, reuse, and contribute back to the system
```

## Initial Capabilities

- Source set to podcast workflow
- NotebookLM audio to publishable media workflow
- Podcast-to-video packaging
- Interview format generation, including future "Major interviewed by his own archive" mode
- Long-form to clips, show notes, summaries, titles, and social assets
- Voice, image, and avatar asset handling as future studio capabilities
- Publishing prep for YouTube, RSS, Substack, short-form, and client delivery
- Sync points for Airtable, Notion, Obsidian, Graphify, and the broader command center
